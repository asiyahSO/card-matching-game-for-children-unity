# Card-matching Game for Children (Unity)
> This project presents an educational mobile card-matching game designed to help children learn basic Japanese vocabulary in a fun and interactive way. The game focuses on three topics: colour, animals, and fruits, and is designed for early learners or beginners in the Japanese language. The game enhances memory, concentration, and language skills by allowing users to match Japanese words with their English equivalents within a time limit. It uses the Fisher-Yates Shuffle Algorithm to randomize cards and ensure each game session is unique. The app offers multiple difficulty levels, visual guidance, and randomized win/lose feedback, promoting active learning through play. This project integrates game-based learning with language acquisition, making vocabulary memorization more effective and enjoyable.
